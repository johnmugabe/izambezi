Installation

CDN (Recommended)

The quickest way to get started. Add this to your HTML <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/izambezi@latest/dist/izambezi.min.css">

NPM

Install via npm for modern build workflows:

npm install izambezi

Then import in your CSS or JavaScript:

// In JavaScript/TypeScript
import 'izambezi/dist/izambezi.min.css';

// Or in SCSS
@import 'izambezi/src/scss/izambezi';

Download

Download the CSS file directly and include it in your project:

<link rel="stylesheet" href="path/to/izambezi.min.css">

Basic Usage

Add iZambezi icons using the <i> tag with the appropriate classes:

<i class="izambezi econet-zw"></i>
<i class="izambezi ecocash"></i>
<i class="izambezi cbz"></i>

Class Structure

Class Description
.izambezi Base class (required) - default size
.[icon-name] Specific icon class (e.g., .econet-zw)

Sizes

Control icon size using size modifier classes:

<i class="izambezi-sm muzinda"></i>   <!-- Small -->
<i class="izambezi muzinda"></i>      <!-- Default -->
<i class="izambezi-lg muzinda"></i>   <!-- Large -->
<i class="izambezi-xl muzinda"></i>   <!-- Extra Large -->
<i class="izambezi-2x muzinda"></i>   <!-- 2x -->
<i class="izambezi-3x muzinda"></i>   <!-- 3x -->

Animations

Add animations using the .animated class along with an animation name:

<!-- Basic animation -->
<i class="izambezi econet-zw animated pulse"></i>

<!-- Infinite animation -->
<i class="izambezi econet-zw animated infinite bounce"></i>

<!-- With speed modifier -->
<i class="izambezi econet-zw animated pulse faster"></i>

<!-- With delay -->
<i class="izambezi econet-zw animated fadeIn delay-2s"></i>

Transforms

Rotate and flip icons using transform classes:

<i class="izambezi zw-bird"></i>
<i class="izambezi zw-bird izambezi-rotate-90"></i>
<i class="izambezi zw-bird izambezi-rotate-180"></i>
<i class="izambezi zw-bird izambezi-flip-horizontal"></i>

Spinning Icons

<i class="izambezi econet-zw izambezi-spin"></i>
<i class="izambezi econet-zw izambezi-pulse"></i>

JavaScript API

For advanced use cases, iZambezi provides an optional JavaScript API:

import IZambezi from 'izambezi';

// Create an instance
const iz = new IZambezi();

// Get all available icons
const icons = iz.getIcons();

// Search icons
const banks = iz.searchIcons('bank');
const universities = iz.getIconsByCategory('university');

// Create an icon element programmatically
const icon = iz.createIcon('econet-zw', {
  size: 'lg',
  animated: true,
  animation: 'pulse',
  infinite: true
});
document.body.appendChild(icon);

// Get HTML string
const html = iz.createIconHTML('ecocash', { size: 'xl' });

// Auto-replace data attributes
// <span data-izambezi="cbz" data-size="lg" data-animated></span>
iz.init();

Data Attribute Usage

You can use data attributes for declarative icon insertion:

<!-- Basic usage -->
<span data-izambezi="econet-zw"></span>

<!-- With options -->
<span data-izambezi="ecocash" 
      data-size="lg" 
      data-animated 
      data-animation="pulse"
      data-infinite></span>

<script>
  const iz = new IZambezi();
  iz.init(); // Replaces all data-izambezi elements
</script>

Browser Support

iZambezi.css works in all modern browsers and IE9+:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)

Animations respect prefers-reduced-motion for accessibility.
